
## Project Overview

This project is a Selenium automation framework for testing an e-commerce website. It uses Java, TestNG, and Maven and follows the Page Object Model design.

## Tech Stack

Java
Selenium WebDriver
TestNG
Maven

## Project Structure

src/test/java

* base
* pages
* tests
* utils
* listeners

src/test/resources

* config.properties

Other folders

* screenshots
* test-output

## Test Scenarios

Login and Registration
Product Search
Add to Cart
Remove from Cart
Checkout

## Features

Page Object Model (POM)
Config-based data (no hardcoding)
Screenshot on failure
TestNG execution

## How to Run

mvn clean test

## Output

Screenshots → screenshots/
Reports → test-output/
